# revolut-exchange-app

A currency exchange web app based on Revolut's app.

Live: https://revolut-exchange-app.vercel.app/

Stack:

- React
- Typescript
- Redux
- React-Query
- Styled Components
- Jest

API: https://openexchangerates.org/

### How to run

```bash
$ yarn
# or
$ yarn start
```

### Tests

```bash
$ yarn test
```
